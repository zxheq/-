5、JDK、JRE、JVM的关系
	JDK = JRE +开发集工具
	JRE = JVM + Java SE 标准类库

6、path环境变量原理
	在path环境变量配置后，执行文件时如果在用户路径下找不到该文件，就会跳转到path环境变量下的目录寻找。

7、文档注释解析命令行
	javadoc -d [新建文件夹名] -author -version [.java文件]

注意：
	①一个文件中最多只能有一个类声明public的，要加也只能加到与文件明同名的类上。
	②编译的过程：
		编译后会生成一个或多个字节码文件，字节码文件的文件名与java源文件中的类名相同。
